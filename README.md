# count-cars-python
Simple cars counter with haarcascades (cars.xml) (Python, CV2)

#### Model
Cars [XML](https://github.com/opencv/opencv/blob/master/data/haarcascades/haarcascade_car.xml)

#### Installation

```bash
pip install -r requirements.txt
```

```bash
python count_cars.py
```
